========================================
GIT COMMANDS CHEAT SHEET
========================================
Обновлено: 02.05.2026

[1] НАЧАЛЬНАЯ НАСТРОЙКА (разово для нового ПК/проекта)
git init                          # Инициализация репозитория в текущей папке
git config --global user.name "Имя"      # Указать имя автора коммитов
git config --global user.email "email"   # Указать email автора
git remote add origin <URL>       # Привязать удалённый репозиторий (GitHub)
git branch -M main                # Переименовать главную ветку в main

[2] ЕЖЕДНЕВНАЯ РАБОТА (основной цикл)
git status                        # Посмотреть статус: какие файлы изменены/добавлены
git add .                         # Добавить ВСЕ изменённые файлы в индекс
git add <файл1> <файл2>           # Добавить только указанные файлы
git commit -m "сообщение"         # Сохранить снимок состояния с комментарием
git push                          # Отправить коммиты на GitHub (в ветку main)
git pull                          # Скачать изменения с GitHub (если работаешь с другого ПК)

[3] ПРОСМОТР И ИСТОРИЯ
git log                           # Показать историю коммитов (q для выхода)
git log --oneline                 # Краткая история в одну строку
git diff                          # Показать изменения в файлах до коммита

[4] ПОЛЕЗНОЕ
git --version                     # Проверить версию Git
git remote -v                     # Проверить привязку к удалённому репозиторию
git stash                         # Временно убрать изменения, не делая коммит

========================================
ПРИМЕЧАНИЯ:
- Токен GitHub (ghp_...) вставляется вместо пароля при git push/pull
- Точка в git add . означает "все файлы в текущей директории"
- Всегда пиши понятные комментарии в git commit -m "..."
========================================

[5] .GITIGNORE (исключение файлов)
cat > .gitignore << 'EOFIGNORE'
*.bak
.DS_Store
__pycache__/
venv/
.env
EOFIGNORE
git rm --cached <файл>        # Удалить файл из Git, но оставить на диске
git add .gitignore
git commit -m "Add .gitignore"
